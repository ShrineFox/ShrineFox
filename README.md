### Hello! I'm ShrineFox 👋
I'm a developer that likes using software to modify videogames. I enjoy sharing my knowledge with the greater community.  
Most of my projects are open-source and I mainly use C#.

### Current Focus ⚡
- I'm creating a mod for Persona 5 Royal (PC) [featuring Vinesauce](https://shrinefox.github.io/en/vinesauce). It's my most ambitious mod since the [Adachi mod for P5 PS3](https://shrinefox.github.io/en/adachi).
  - I will be sharing tools that I've made for myself to help create this project. See my [Trello](https://trello.com/b/mXTXRmpR/status) for more information.
  - I am considering porting the Adachi mod to P5R PC afterward.
- I also maintain the [Persona 5 Mod Menu](https://github.com/ShrineFox/Persona-5-Mod-Menu), an in-game custom script that lets you freely make use of native game functions.
- My website, [shrinefox.com](https://shrinefox.com), is an ever-evolving knowledge base for related subjects (featuring news, a wiki, some web apps...).
- Eventually, I intend to complete an all-in-one workspace solution for modding Persona games (called [PStudio](https://github.com/shrinefox/P-Studio)).
- I also have a quality-of-life and [in-game mod menu project](https://github.com/ShrineFox/EarthBound-Mod-Menu) brewing for SNES cult classic EarthBound, which I want to finish and release soon as well.

  ### More Information 💬
  As you can see, I have a wide range of ambitions. While I'd like to give them all equal attention, I can only make meaningful progress by focusing on one at a time.  
  Please consider [sending me a tip on Ko-Fi](https://ko-fi.com/shrinefox) if any of my work has benefitted you! ❤
